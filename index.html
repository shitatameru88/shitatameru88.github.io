Getting started

How to enable
User Sources

Sources are off by default. Here's how to turn them on and start reading.

1

Open the More tab

Tap the More tab in the bottom navigation bar (the last icon, bottom-right). This is where Settings, sources, and all app controls live.

2

Tap User Sources

At the top of the More screen, tap User Sources. You'll see a list of all available source adapters built into the app, along with their health status.

3

Toggle a source on

Find the source you want and flip its toggle. A confirmation dialog will appear — tap Enable to confirm. The source will start a health check immediately. A green indicator means it's reachable.

4

Go to Browse and search

Head to the Browse tab. The search bar will now say "Search in [Source Name]". Type any title to start reading. Save titles to your library from the manga detail screen.

5

Save & track updates

Tap Save to Library on any manga to follow it. The Updates tab shows new chapters grouped by day. Pull down to check for the latest chapters across all saved titles at once.

Note: Sources connect to third-party websites. Availability and content depend on those services — MangaRead does not host, distribute, or affiliate with any source content. Only enable sources you are permitted to access in your region.
